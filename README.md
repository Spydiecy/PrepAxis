# PrepAxis — AI Interview Coach

An intelligent interview preparation platform that helps users improve their interview skills through AI-powered feedback, resume analysis, and mock interviews.

## Features

- 🎤 **Voice & Text Interviews** - Practice interviews with real-time feedback
- 📝 **Resume Analysis** - Get personalized resume improvement suggestions
- 📊 **Performance Tracking** - Track progress with detailed interview reports
- 👤 **User Authentication** - Secure login and registration
- 💬 **Interactive Chat** - Real-time conversation analysis

## Getting Started

### Prerequisites

- Node.js (v14+)
- npm

### Installation

```bash
npm install
```

### Available Scripts

```bash
npm start      # Start development server (http://localhost:3000)
npm test       # Run tests
npm run build  # Build for production
```

## Project Structure

```
src/
├── pages/              # Route pages
│   ├── auth/          # Login & Register
│   ├── interview/     # Interview pages
│   ├── resume/        # Resume pages
│   └── Dashboard.jsx
├── components/         # Reusable components
│   ├── interview/     # Interview UI components
│   ├── layout/        # Layout components
│   └── resume/        # Resume components
└── App.jsx            # Main router
```

## Tech Stack

- React 19
- React Router 7
- Tailwind CSS 4
- Lucide React (Icons)
